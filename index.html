📗 Google Sheets Koppeling
Sync je trades rechtstreeks naar Google Sheets via de Sheets API. Eenmalig een API key instellen — verloopt nooit.
⚠️ Nog niet gekoppeld aan Google Sheets
Setup in 2 stappen
1
Ga naar console.cloud.google.com/apis/credentials → kopieer je API key en plak hieronder.
2
Zorg dat je Google Sheet "Trades" gedeeld is op "Iedereen met de link kan bewerken" via de Share knop.
🔒 Key wordt alleen lokaal opgeslagen in jouw browser
Gebruik je bestaande Apps Script deployment URL
📥 Excel Importeren
📂
Importeer je bestaande A_WHEEL_TRACKER.xlsx rechtstreeks. De app leest het Trades tabblad uit en laadt alle trades in één klik.
Bestand wordt lokaal verwerkt — er gaat niets naar het internet
📱 Installeer als App
📱 iPhone (Safari):
Stap 1: Open henckt-hue.github.io/Wheel-G in Safari
Stap 2: Tik op het deel-icoon ↑ onderaan
Stap 3: Tik op "Zet op beginscherm"
Stap 4: Naam: Wheel Pro → tik "Voeg toe"

💻 Desktop (Chrome/Edge):
Klik op de knop hierboven, of klik op het install-icoon (⊕) in de adresbalk van je browser.

De app werkt volledig scherm, zonder browser-balk, met het Wheel Pro icoon.
